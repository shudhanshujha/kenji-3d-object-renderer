# Kenji 3D Object Renderer 🚀

A high-performance, interactive 3D object renderer built with Python, featuring custom projection logic for real-time visualization of 3D models and primitives.

## 🌟 Key Features
- **Custom 3D Engine:** Built from scratch using NumPy for mathematical projections and Tkinter for rendering.
- **Interactive Camera:** Smooth mouse-based rotation, panning, and zooming for a seamless 3D experience.
- **Diverse Primitives:** Instant generation of 3D shapes including Spheres, Cubes, Toruses, Pyramids, and more.
- **Model Loader:** Support for loading and viewing external `.obj` files.
- **Multiple Render Modes:** Toggle between Solid, Wireframe, and Points views.
- **Web Interface:** Includes a Gradio-powered web application for browser-based 3D model previewing.

## 🛠️ Tech Stack & Tools
- **Language:** Python 3.x
- **Core Library:** Tkinter (GUI & Canvas Rendering)
- **Math Library:** NumPy (Matrix transformations & Projections)
- **Web Framework:** Gradio (Web App Interface)
- **Development Tools:** Git, VS Code, Python

## 🚀 Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/kenji-3d-object-renderer.git
   cd kenji-3d-object-renderer
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application:**
   - **Desktop Version:**
     ```bash
     python main.py
     ```
   - **Web Preview:**
     ```bash
     python web_app.py
     ```

## 🎮 Controls
- **Left Mouse Button (LMB):** Rotate the 3D model.
- **Scroll Wheel:** Zoom in and out of the viewport.
- **Control Panel:** Select primitives, change render modes, or toggle auto-rotation.
